# Asset Class Trend Following 策略重現報告 (2025)

## 策略說明
本策略採用 Asset Class Trend Following 方法，針對 **2025 年度資料** 進行回測。核心邏輯在於選取價格高於移動平均線（SMA）且動能（ROC）為正的資產，並從中選取動能最強的前 3 檔進行投資。

## 核心參數
沿用 2024 年螞蟻演算法（ACO）最佳化後選定的參數：
- **SMA 週期**: 69
- **ROC 週期**: 23
- **停損比例 (StopLoss%)**: 9.0% (最高價回落停損)

## 績效表現 (2025)
- **CAGR (年化報酬率)**: 100.40%
- **MaxDD (最大回撤)**: -37.90%
- **Calmar Ratio**: 2.65
- **Win Rate (勝率)**: 51.85%

## 策略執行規則
1. **初始資金**: 30,000,000 元。
2. **再平衡**: 每 5 個交易日進行一次再平衡。**信號產生於 T 日，執行於 T+1 日。**
3. **選股**: 
   - 價格 > SMA
   - ROC > 0
   - 取 ROC 前 3 名。
4. **持倉**: 若原持股仍在前 3 名，則繼續持有（保持原股數）。**保留持股規則優先於所有其他交易邏輯。**
5. **停損**: 當收盤價較持有期間最高收盤價下跌超過 9.0% 時觸發停損。**停損信號僅在再平衡日進行檢查與執行，以確保 5 日週期的穩定性。**
6. **執行**: T 日產生訊號，T+1 日以收盤價執行。

## 結論
本策略在 2025 年回測期間表現如下：CAGR 達到 100.40%，獲利能力卓越。策略核心在於嚴格執行「保留持股」優先邏輯，有效降低了在市場波動期間的週轉成本。雖然 MaxDD 較大，但透過 5 日週期的過濾與趨勢跟蹤，成功捕捉了全年的多頭趨勢。
