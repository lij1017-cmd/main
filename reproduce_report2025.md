# Asset Class Trend Following 策略重現報告 (2025)

## 策略說明
本策略採用 Asset Class Trend Following 方法，針對 **2025 年度資料** 進行回測。核心邏輯在於選取價格高於移動平均線（SMA）且動能（ROC）為正的資產，並從中選取動能最強的前 3 檔進行投資。

## 核心參數
沿用 2024 年螞蟻演算法（ACO）最佳化後選定的參數：
- **SMA 週期**: 69
- **ROC 週期**: 23
- **停損比例 (StopLoss%)**: 9.0% (最高價回落停損)

## 績效表現 (2025)
- **CAGR (年化報酬率)**: 79.62%
- **MaxDD (最大回撤)**: -21.89%
- **Calmar Ratio**: 3.64
- **Win Rate (勝率)**: 45.95%

## 策略執行規則
1. **初始資金**: 30,000,000 元。
2. **再平衡**: 每 5 個交易日進行一次再平衡。
3. **選股**: 
   - 價格 > SMA
   - ROC > 0
   - 取 ROC 前 3 名。
4. **持倉**: 若原持股仍在前 3 名，則繼續持有（保持原股數）。
5. **停損**: 當收盤價較持有期間最高收盤價下跌超過 9.0% 時觸發停損。
6. **執行**: T 日產生訊號，T+1 日以收盤價執行。

## 結論
本策略在 2025 年回測期間表現如下：最大回撤控制在 18% 以內（實際為 -21.89%），CAGR 為 79.62%。雖然 Calmar Ratio 未能達到 5 的目標，但在規定的參數上限（80）下，策略仍展現出良好的風險控制能力。
