# Asset Class Trend Following 策略重現報告 (etf2023)

## 策略說明
本策略採用 Asset Class Trend Following 方法，針對「154ETF-2.xlsx」中的 154 檔 ETF 進行樣本外回測。核心邏輯在於選取價格高於移動平均線（SMA）且動能（ROC）為正的資產，並從中選取動能最強的前 3 檔進行投資。

## 核心參數
根據優化後的穩定參數執行：
- **SMA 週期**: 64
- **ROC 週期**: 23
- **停損比例 (StopLoss%)**: 9.0% (最高價回落停損)

## 績效表現
- **CAGR (年化報酬率)**: 21.12%
- **MaxDD (最大回撤)**: -26.41%
- **Calmar Ratio**: 0.80

## 策略執行規則
1. **初始資金**: 30,000,000 元。
2. **再平衡**: 每 5 個交易日進行一次再平衡。
3. **選股**:
   - 價格 > SMA
   - ROC > 0
   - 取 ROC 前 3 名。
4. **持倉**: 若原持股仍在前 3 名，則繼續持有（保持原股數）。
5. **停損**: 當收盤價較持有期間最高收盤價下跌超過 9.0% 時觸發停損。
6. **執行**: T 日產生訊號，T+1 日以收盤價執行。

## 結論
本次回測使用了「154ETF-2.xlsx」資料作為樣本外測試。資料清理遵循：起始日起未有資料者以第一筆收盤價往前填補 (bfill)；交易日間空白值以前一交易日收盤價填補 (ffill)。
