# Asset Class Trend Following 策略重現報告 (2025 - 納入交易成本)

## 策略說明
本策略採用 Asset Class Trend Following 方法，並納入交易成本（買進 0.1425%，賣出 0.1425% + 0.3% 證交稅），針對「個股合-1.xlsx」中的商品進行回測。核心邏輯在於選取價格高於移動平均線（SMA）且動能（ROC）為正的資產，並從中選取動能最強的前 3 檔進行投資。

## 核心參數
- **SMA 週期**: 64
- **ROC 週期**: 23
- **停損比例 (StopLoss%)**: 9.0% (最高價回落停損)

## 績效表現
- **CAGR (年化報酬率)**: 27.71%
- **MaxDD (最大回撤)**: -18.33%
- **Calmar Ratio**: 1.51

## 策略執行規則
1. **初始資金**: 30,000,000 元。
2. **再平衡**: 每 5 個交易日進行一次再平衡。
3. **選股**:
   - 價格 > SMA
   - ROC > 0
   - 取 ROC 前 3 名。
4. **持倉**: 若原持股仍在前 3 名，則繼續持有（保持原股數）。
5. **停損**: 當收盤價較持有期間最高收盤價下跌超過 9.0% 時觸發停損。
6. **執行**: T 日產生訊號，T+1 日以收盤價執行。

## 結論
本次回測使用了「個股合-1.xlsx」資料，並將 SMA 調整為 64，ROC 調整為 23。
