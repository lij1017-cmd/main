# Asset Class Trend Following 策略重現報告 (2025-1)

## 策略說明
本策略採用 Asset Class Trend Following 方法，針對 **2025 年度資料** 進行回測。本次回測使用調整後的參數：SMA = 64。核心邏輯在於選取價格高於移動平均線（SMA）且動能（ROC）為正的資產，並從中選取動能最強的前 3 檔進行投資。

## 核心參數
調整後的參數值：
- **SMA 週期**: 64
- **ROC 週期**: 23
- **停損比例 (StopLoss%)**: 9.0% (最高價回落停損)

## 績效表現 (2025-1)
- **CAGR (年化報酬率)**: 55.38%
- **MaxDD (最大回撤)**: -38.95%
- **Calmar Ratio**: 1.42
- **Win Rate (勝率)**: 52.24%

## 策略執行規則
1. **初始資金**: 30,000,000 元。
2. **再平衡**: 每 5 個交易日進行一次再平衡。**交易信號於 T 日收盤後產生，並於 T+1 日以收盤價執行。**
3. **選股**:
   - 價格 > SMA
   - ROC > 0
   - 取 ROC 前 3 名。
4. **持倉**: 若原持股仍在前 3 名，則在再平衡日執行「保持」(保持原股數)。
5. **停損**: 當收盤價較持有期間最高收盤價下跌超過 9.0% 時觸發停損。**停損規則為每日檢查，一旦觸發，於次一交易日 (T+1) 以收盤價賣出。**
6. **執行**: 所有交易動作均遵循 T 日訊號、T+1 日執行之原則。

## 結論
本策略在 2025-1 回測期間表現如下：CAGR 為 55.38%，最大回撤為 -38.95%。相較於原先最佳化參數 (SMA=69)，調低 SMA 週期至 64 導致了績效的小幅下降。參數高原表顯示在 60 至 68 區間內，策略表現相當穩定，顯示該參數區域的穩健性。
