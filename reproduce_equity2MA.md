# Asset Class Trend Following (equity-2MA) 策略說明文件

## 1. 策略概述
本策略採用 **Asset Class Trend Following** 方法，針對 131 檔商品進行回測。核心邏輯結合了雙移動平均線 (Double SMA) 的趨勢過濾、ROC 動能指標，以及多重流動性與價格趨勢濾網。策略旨在最大化風險調整後報酬（Calmar Ratio），並透過螞蟻演算法（ACO）對參數進行全域最佳化。

## 2. 核心參數與最佳化結果
經過 2019-2023 年的樣本集最佳化，並在 2024-2025 年的測試集驗證，最終選定之最適參數如下：

- **SMA 1 (短期均線):** 150
- **SMA 2 (長期均線):** 345
  - *進場基準均線：(SMA150 + SMA345) / 2*
- **ROC 週期 (Momentum):** 17
- **停損機制:** 移動平均線停損 (MA Stop)
  - *停損參考值：15 日均線*
- **再平衡週期:** 9 個交易日

## 3. 交易規則說明
### 3.1 進場邏輯
1. **趨勢過濾:** 當標的價格高於雙均線之平均值 `(SMA150 + SMA345) / 2`。
2. **動能過濾:** 當標的 17 日 ROC (Rate of Change) 為正值。
3. **流動性濾網:** 排除當日成交金額低於 **3000 萬** TWD 之標的。
4. **趨勢一致性:** 標的價格需同時高於其自身的 **5日、10日、20日均線**。
5. **標的選取:** 在符合上述所有條件的標的中，選取 **ROC 數值最高的前 3 名** 進行投資。

### 3.2 交易執行
- **T 日產生訊號，T+1 日以收盤價執行。**
- **持倉管理:** 若原有持股仍位列前 3 名且符合濾網條件，則續抱；若跌出排名或不符條件，則於再平衡日賣出，並換入新訊號。

### 3.3 停損機制
- **MA Stop:** 當收盤價跌破 **15 日均線** 時，立即於次一交易日觸發停損賣出，不等待再平衡日。

### 3.4 資金管理 (動態分配 V2)
- **初始資金:** 30,000,000 元。
- **槽位機制:** 投資組合分為 3 個槽位，每槽位預算上限為 1,000 萬。
- **所得再投:** 賣出所得優先作為該槽位新訊號的預算，超出 1,000 萬的部分回流至中央資金池。

## 4. 績效表現 (2019-2025 全週期)
- **年化報酬率 (CAGR):** 32.83%
- **最大回撤 (MaxDD):** -21.16%
- **Calmar Ratio:** 1.55
- **勝率 (Win Rate):** ~45% (視交易對而定)

### 分段表現
- **樣本集 (2019-2023):** CAGR 30.68%, MaxDD -21.16%, Calmar 1.45
- **測試集 (2024-2025):** CAGR 36.59%, MaxDD -5.82%, Calmar 6.29
