# Asset Class Trend Following 策略重現報告 (2024)

## 策略說明
本策略採用 Asset Class Trend Following 方法，針對 129 檔商品進行回測。

## 核心參數
- **SMA 週期**: 69
- **ROC 週期**: 23
- **停損比例 (StopLoss%)**: 9.0%

## 績效表現
- **CAGR (年化報酬率)**: 34.86%
- **MaxDD (最大回撤)**: -11.70%
- **Calmar Ratio**: 2.98
- **Win Rate (勝率)**: 48.09%

## 策略執行規則
1. **初始資金**: 30,000,000 元。
2. **再平衡**: 每 5 個交易日進行一次再平衡。
3. **選股**: 價格 > SMA, ROC > 0, 取 ROC 前 3 名。
4. **持倉**: 若可用現金不足 1,000 萬，則不買入新標的。
5. **停損**: 收盤價較持有期間最高收盤價下跌超過 9.0% 時觸發停損。

## 策略驗證
### 2019/05/02 持股說明
當日符合標準的前 3 名動能值分別為 41.74%, 28.99%, 25.93%。
由於賣出前期標的後，可用現金約為 952.9 萬，低於買入新標的所需的 1,000 萬門檻，故僅買入動能排名更高的標的，該日僅持有 2 檔。
