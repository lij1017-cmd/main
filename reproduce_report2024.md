# Asset Class Trend Following 策略重現報告 (2024)

## 策略說明
本策略採用 Asset Class Trend Following 方法，針對 129 檔商品進行回測。核心邏輯在於選取價格高於移動平均線（SMA）且動能（ROC）為正的資產，並從中選取動能最強的前 3 檔進行投資。

## 核心參數
經過螞蟻演算法（ACO）最佳化後，選定以下參數：
- **SMA 週期**: 69
- **ROC 週期**: 23
- **停損比例 (StopLoss%)**: 9.0% (最高價回落停損)

## 績效表現
- **CAGR (年化報酬率)**: 34.84%
- **MaxDD (最大回撤)**: -11.70%
- **Calmar Ratio**: 2.98
- **Win Rate (勝率)**: 48.09%

## 策略執行規則
1. **初始資金**: 30,000,000 元。
2. **再平衡**: 每 5 個交易日進行一次再平衡。
3. **選股**:
   - 價格 > SMA
   - ROC > 0
   - 取 ROC 前 3 名。
4. **持倉**: 若原持股仍在前 3 名，則繼續持有（保持原股數）。
5. **停損**: 當收盤價較持有期間最高收盤價下跌超過 9.0% 時觸發停損。
6. **執行**: T 日產生訊號，T+1 日以收盤價執行。

## 策略驗證與說明

## 結論
本策略在回測期間表現穩健，成功將最大回撤控制在 18% 以內（實際為 11.70%）。雖然 Calmar Ratio 未能達到 5 的目標，但在規定的參數上限（80）下已展現出優異的風險控制能力。
