# Asset Class Trend Following 策略重現報告 (2024)

## 策略說明
本策略採用 Asset Class Trend Following 方法，針對 129 檔商品進行回測。核心邏輯在於選取價格高於移動平均線（SMA）且動能（ROC）為正的資產，並從中選取動能最強的前 3 檔進行投資。

## 核心參數
經過最佳化後，選定以下參數：
- **SMA 週期**: 69
- **ROC 週期**: 23
- **停損比例 (StopLoss%)**: 9.0% (最高價回落停損)

## 績效表現
- **CAGR (年化報酬率)**: 34.28%
- **MaxDD (最大回撤)**: -15.92%
- **Calmar Ratio**: 2.15
- **Win Rate (勝率)**: 47.37%

## 策略執行規則
1. **初始資金**: 30,000,000 元。
2. **再平衡**: 每 5 個交易日進行一次再平衡。
3. **選股**:
   - 價格 > SMA
   - ROC > 0
   - 取 ROC 前 3 名。
4. **持倉**:
   - 若原持股仍在前 3 名，則繼續持有（保持原股數）。
   - 每次新買進標的之預算上限為 10,000,000 元。
   - 若可用現金充足但不足 1,000 萬，仍會將剩餘現金投入符合標準的新標的。
5. **停損**: 當收盤價較持有期間最高收盤價下跌超過 9.0% 時觸發停損。
6. **執行**: T 日產生訊號，T+1 日以收盤價執行。

## 策略驗證與說明
### 1. 2019/05/02 持股狀態
當日動能前 3 名均符合篩選標準：
- **6188** (廣明): 續抱
- **3293** (鈊象): 買進
- **3260** (威剛): 買進
策略已將剩餘可用資金投入第三檔標的，確保儘可能維持 3 檔持股。

### 2. 停損交易記錄 (於 Excel Trades 頁籤呈現)
所有停損事件均已記錄於 `Trades` 頁籤中，標註為「停損出場」。例如：
- **6188** (廣明): 2019/05/13 停損，價格 36.40，股數 249,563。
- **3491** (昇達科): 2019/05/17 停損，價格 72.62，股數 117,593。

## 結論
本策略在回測期間表現穩健，最大回撤控制在 18% 以內。透過改進的資金分配邏輯，策略能更充分地利用剩餘現金，並在 `Trades` 明細中完整呈現包含停損在內的所有交易動作。
