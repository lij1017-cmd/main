# B303 策略優化效應分析報告 (PART 2)

本報告針對 B303 策略優化後的各項因子（市場寬度濾網、Volatility 停損）進行單獨效應分析，並提供成本壓力測試結果。

---

## 1. 各因子效應比較表
針對各因子單獨加入與原始基準策略之比較：

| 指標 | 基準 (原始 equityV) | 方案 A (+濾網) | 方案 B (+Vol 停損) | 方案 C (完整優化) |
| :--- | :--- | :--- | :--- | :--- |
| **SMA** | 303 | 303 | 303 | 303 |
| **ROC** | 14 | 14 | 14 | 14 |
| **停損方式** | 固定 9.99% | 固定 10% | **Vol 停損** | **Vol 停損** |
| **市場寬度濾網** | 無 | **有** | 無 | **有** |
| **CAGR (30M)** | 32.69% | 33.73% | 32.30% | 33.66% |
| **MaxDD** | -11.22% | -11.22% | -10.32% | -10.28% |
| **Calmar** | 2.91 | 3.01 | 3.13 | 3.28 |
| **2022 年報酬** | -1.50% | +1.78% | -2.98% | +1.81% |

### 分析結論：
1. **市場寬度濾網 (方案 A)**：顯著改善了 2022 年空頭市場的表現，使報酬由負轉正 (+1.78%)，並提升了整體 CAGR 與 Calmar 比率。
2. **Volatility 停損 (方案 B)**：主要貢獻在於風險控管，將 MaxDD 從 -11.22% 降至 -10.32%，雖然單獨使用時 CAGR 略微下降，但風險報酬比 (Calmar) 得到提升。
3. **完整方案 (方案 C)**：結合兩者優點，實現了最高的風險報酬比 (Calmar 3.28)，同時在 2022 年維持正報酬 (+1.81%)，且 MDD 控制在最優水平 (-10.28%)。

## 2. 成本壓力測試分析表
評估在不同滑價情境下，各方案的穩定性：

| 成本假設情境 | 基準 (原始) | 方案 A | 方案 B | 方案 C |
| :--- | :--- | :--- | :--- | :--- |
| **標準** (僅含手續費+稅) | 32.69% / -11.22% | 33.73% / -11.22% | 32.30% / -10.32% | 33.66% / -10.28% |
| **保守** (停損額外 +0.3%) | 32.21% / -11.52% | 33.33% / -11.52% | 31.84% / -10.74% | 33.30% / -10.50% |
| **極端** (濾網清倉額外 +0.5%)| 32.69% / -11.22% | 33.51% / -11.22% | 32.30% / -10.32% | 33.45% / -10.28% |

*(格式說明：CAGR (30M) / MaxDD)*

### 分析結論：
1. **耐受度高**：即便在保守情境下（停損滑價 0.3%），方案 C 的 CAGR 仍維持在 33% 以上，顯示策略具備足夠的獲利緩衝。
2. **濾網穩定性**：極端情境（濾網清倉滑價 0.5%）對方案 C 的影響極小（CAGR 僅微降至 33.45%），這歸功於市場寬度濾網的觸發頻率適中，並未造成過度的過度交易成本。
3. **推薦選擇**：方案 C 在所有成本情境下均表現出最優的 Calmar 比率與回撤控制能力，為目前最理想的優化版本。
