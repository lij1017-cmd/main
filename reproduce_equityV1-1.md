# Asset Class Trend Following 策略回測報告 (equityV1-1)

## 策略說明
本報告彙整使用參數 **SMA=303, ROC=14, Stop Loss=9.99%, Rebalance=9天** 的回測結果。
此版本包含 2019.01.01 - 2026.03.31 的全期間回測與多區間 Walk-Forward Analysis (WFA)。

- **參數設定**：
  - SMA 週期：303
  - ROC 週期：14
  - 停損比例：9.99% (最高價回落)
  - 再平衡週期：9 天
